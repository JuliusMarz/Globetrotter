# Globetrotter
This is some code to create your own "Geoguessr" style tournament with your friends.
!!It only works properly, if you have downloaded all the contents in this folder!!

Rankings.html will give you a nice overview over the current rankings

Round1.html will give you results for Round 1
Round2.html will give you results for Round 2
Roud3html...

The different jpeg files show the pictures (+ bonusviews) for every round.

All the .Rmd files are here to modify the content by your own will. I am happy for any updates that people want to make. This is a collaborative space!
